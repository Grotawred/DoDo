# 📝 DoDo — ToDo List

**DoDo** — це зручний та мінімалістичний сервіс для планування завдань, розроблений для тих, хто цінує простоту та ефективність.

## 🎯 Тема проєкту
Створення зручного мінімалістичного ToDo списку завдань.

## 💻 Програмне забезпечення
Під час розробки використовувалися наступні інструменти:
* **IDE:** IntelliJ IDEA Community Edition, Visual Studio Code
* **Робота з БД:** MySQL Workbench, [drawsql.app](https://drawsql.app)
* **Дизайн:** Figma
* **Сервер:** AWS, DuckDNS
* **Контейнер:** Docker, Terraform

## 🛠 Мови програмування
* **Backend:** Java / Spring Boot Framework
* **Frontend:** HTML / CSS

## 📄 Опис проєкту
Додаток має чітку структуру та складається з наступних модулів:

### 1. Авторизація та вхід
* **Початкова сторінка:** Містить кнопки для швидкого переходу до реєстрації або входу в акаунт.
* **Сторінки реєстрації/входу:** Лаконічні форми для створення нового профілю або доступу до існуючого.

### 2. Робочий простір
Головна сторінка додатка розділена на дві ключові зони:
* **Бічна панель (Sidebar):** * Інструменти для створення нових категорій (списків).
    * Зручна фільтрація завдань за списками.
* **Основна частина (Content Area):**
    * Відображення завдань вибраного списку.
    * Форма для швидкого створення нових завдань.
    * Функціонал для видалення завершених або неактуальних пунктів.

## 🎨 Дизайн та інтерфейс
Проєкт виконаний у **світлих тонах**, що забезпечує легкість сприйняття та фокус на головному. Інтерфейс побудований на сучасних CSS-стилях (Flexbox/Grid) із використанням плавних переходів та м'яких тіней.
